Create a new Target Node Profile
  • A Target Node Profile controls the list of EC2 instance types, maximum storage size as well as approved subnet ids for your Target Node Software Stack.
  • You can have multiple Software Stacks using the same Target Node Profile.
Note:
  • Maximum EBS storage the user will be able to provision for their target node
Note:
  • Support Exact Instance Type (c6i.xlarge)
  • Support Instance Family (c6i.*)
  • Refer to Amazon EC2 Instance types page for a list of EC2 instances
  • You will need to update boto3 to add instances released after the date of your SOCA installation
Note:
  • Subnets must be from the list of private subnets configured on SOCA
