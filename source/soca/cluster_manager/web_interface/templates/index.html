Engineering Development Hub (EDH)

A unified platform for cloud workstations, cloud simulations, and AI-supported engineering workflows.

{% include 'common/flashed_messages.html' %}

Benefits and Features


Virtual Desktops

Secure, scalable cloud-based workstations designed for engineering workloads and heavy compute tasks.


Embeeded Development

Run, test, and emulate embedded ECUs and target hardware directly in the cloud.


High Performance Computing

Easily submit jobs for LSF, OpenPBS, PBSPro, Slurm and seamlessy customize compute, storage and operating systems requirements for your HPC simulations

100% Customizable

EDH is built entirely on top of AWS and can be customized by users as needed. More importantly, the entire codebase is open-source and available on GitHub


Support for network license

EDH includes a Floating Licenses-enabled script which calculate the number of licenses for a given feature and only start to provision capacity when enough licenses are available


Budgets and Cost Management

create, monitor and manage your budgets for your Virtual Targets, Virtual Desktops and Compute Nodes in a single place. Gain full visibility via automatic AWS Cost Allocation tags.
